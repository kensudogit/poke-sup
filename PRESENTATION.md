## スライド 16: クイックスタート

### 🚀 簡単なセットアップ

```bash
# 1. リポジトリのクローン
git clone <repository-url>
cd poke-sup

# 2. Docker Composeで起動
docker-compose up -d

# 3. データベースの初期化
docker-compose exec backend python scripts/init_db.py

# 4. ブラウザでアクセス
# http://localhost:3002
```

**わずか3ステップで起動可能**
