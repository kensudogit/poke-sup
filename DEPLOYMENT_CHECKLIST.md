## 📋 デプロイ手順

### 1. 環境変数の設定

```bash
# .env.production を作成
SECRET_KEY=<強力なランダム文字列>
JWT_SECRET_KEY=<強力なランダム文字列>
DATABASE_URL=<本番データベースURL>
FLASK_ENV=production
```

### 2. データベースマイグレーション

```bash
docker-compose exec backend python scripts/init_db.py
```

### 3. セキュリティ設定

- CORS設定の見直し
- レート制限の有効化
- HTTPS証明書の設定

### 4. デプロイ

```bash
docker-compose -f docker-compose.prod.yml up -d
```
