# ポケさぽ (Poke-Sup) - 医療コミュニケーションプラットフォーム

## 📋 エグゼクティブサマリー

**ポケさぽ**は、患者と医療従事者間のコミュニケーションを革新的に改善する、次世代の医療コミュニケーションプラットフォームです。リアルタイムチャット、健康データ管理、リマインダー機能を統合し、医療サービスの質を向上させ、患者満足度を高めます。

### 🎯 主要な価値提案

- **リアルタイムコミュニケーション**: 患者と医療従事者が即座に連絡を取り合える
- **包括的な健康管理**: 健康データの一元管理と可視化
- **効率的なケア管理**: 自動リマインダーによる服薬・診察の管理
- **セキュアなプラットフォーム**: 医療情報の安全な管理とプライバシー保護
- **スケーラブルなアーキテクチャ**: 成長に応じて拡張可能

---

## 🏥 解決する課題

### 現在の医療現場の課題

1. **コミュニケーションの非効率性**
   - 電話やメールでの連絡が煩雑
   - 緊急時の連絡手段が限定的
   - 会話履歴の管理が困難

2. **健康データ管理の分散**
   - 患者が複数の場所でデータを記録
   - 医療従事者がデータを統合して分析することが困難
   - データの可視化が不十分

3. **ケア継続性の課題**
   - 服薬や診察のリマインダーが不十分
   - 患者の自己管理が困難
   - 治療計画の遵守率が低い

### ポケさぽによる解決

✅ **一元化されたコミュニケーションハブ**: すべての対話を一箇所で管理  
✅ **統合された健康データ管理**: すべての健康情報を一元的に管理・可視化  
✅ **自動化されたリマインダー**: 服薬・診察の管理を自動化  
✅ **リアルタイムの情報共有**: 即座に情報を共有し、迅速な対応を実現

## 🛠️ 技術スタック

### バックエンド
- **言語**: Python 3.11
- **フレームワーク**: Flask (軽量で高速)
- **リアルタイム通信**: Socket.IO (WebSocket)
- **認証**: JWT (JSON Web Tokens)
- **データベース**: PostgreSQL 15 (信頼性の高いRDBMS)

### フロントエンド
- **フレームワーク**: Next.js 14 (React 18)
- **言語**: TypeScript (型安全性)
- **スタイリング**: Tailwind CSS (モダンなUI)
- **状態管理**: Zustand (軽量で効率的)
- **データ可視化**: Recharts (インタラクティブなグラフ)

### インフラストラクチャ
- **コンテナ化**: Docker & Docker Compose
- **スケーラビリティ**: マイクロサービス対応アーキテクチャ
- **デプロイメント**: Railway, Vercel対応

### セキュリティ
- **データ暗号化**: HTTPS/TLS
- **認証**: JWT with secure tokens
- **入力検証**: 包括的なバリデーション
- **SQLインジェクション対策**: ORMによる保護

## 🛠️ セットアップとインストール

### 前提条件

- Docker & Docker Compose
- Node.js 18+
- Python 3.11+ (オプション)

### クイックスタート

```bash
# 1. リポジトリのクローン
git clone <repository-url>
cd poke-sup

# 2. Docker Composeで起動
docker-compose up -d

# 3. データベースの初期化
docker-compose exec backend python scripts/init_db.py

# 4. ブラウザでアクセス
# フロントエンド: http://localhost:3002
# バックエンドAPI: http://localhost:5002
```

### 詳細なセットアップ

詳細なセットアップ手順については、[SETUP.md](./SETUP.md) を参照してください。
