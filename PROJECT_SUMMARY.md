## 🚀 次のステップ

### すぐに始められること

1. **環境変数の設定**
   ```bash
   # バックエンド
   cd backend
   cp .env.example .env
   # .envファイルを編集
   
   # フロントエンド
   cd frontend
   cp .env.example .env.local
   # NEXT_PUBLIC_API_URLを確認
   ```

2. **Docker Composeで起動**
   ```bash
   docker-compose up -d
   ```

3. **ブラウザでアクセス**
   - http://localhost:3000
